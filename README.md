# 🤖 DocGen — Генератор документации с AI

[![Python](https://img.shields.io/badge/python-3.10+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Stars](https://img.shields.io/github/stars/ТВОЙ_НИК/docgen?style=social)](https://github.com/ТВОЙ_НИК/docgen)

**Автоматическая генерация документации для Python проектов за 1-5 минут.**

DocGen анализирует ваш код и создаёт профессиональную README.md с помощью локальных AI-моделей. Полностью бесплатно и приватно!

---

## ✨ Возможности

- 🎯 **3 уровня качества** — выбирай между скоростью и детализацией
- 🆓 **100% бесплатно** — никаких подписок и API ключей
- 🔒 **Приватно** — код не покидает ваш компьютер
- ⚡ **Быстро** — документация за 1-5 минут
- 🌍 **На русском** — понятный интерфейс и документация

---

## 📦 Установка

### 1. Установите Python

Скачайте с [python.org](https://www.python.org/downloads/) (версия 3.10+)

### 2. Установите Ollama

**Обязательно!** Скачайте с [ollama.com](https://ollama.com)

> 💡 Ollama нужна для работы локальных AI-моделей. Это бесплатно и занимает 2 минуты.

### 3. Скачайте DocGen

```bash
git clone https://github.com/Pasha0010/docgen.git
cd docgen
pip install -e .
```
